-------------------------------------- Güncelleme-----------------------------
```
cd
wget https://github.com/pactus-project/pactus/releases/download/v1.3.0/pactus-cli_1.3.0_linux_amd64.tar.gz
tar -xvf pactus-cli_1.3.0_linux_amd64.tar.gz
```
### Screen'e girelim. 
```
screen -r pactus
```
##### Ardından CTRL+C ile durduralım.
```
cd
cd pactus-cli_1.3.0
```
```
./pactus-daemon start -w ~/pactus
```
